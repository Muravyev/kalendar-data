Политика конфиденциальности мобильного приложения «Календарь вакцинации»
1. Общие положения
Настоящая Политика определяет порядок обработки и защиты персональных данных пользователей мобильного приложения «Календарь вакцинации».
2. Перечень обрабатываемых данных
Приложение обрабатывает только те данные, которые пользователь самостоятельно вводит для формирования индивидуального плана вакцинации (дата рождения, сведения о состоянии здоровья и др.).
3. Цели обработки данных
Данные используются исключительно для генерации индивидуального плана вакцинации.
Данные не сохраняются и не передаются третьим лицам.
4. Срок хранения данных
Все введённые данные удаляются сразу после формирования плана.
5. Безопасность данных
Приложение не сохраняет и не передаёт персональные данные, все вычисления производятся локально на устройстве пользователя.
6. Права пользователя
Пользователь вправе не предоставлять данные, а также прекратить использование приложения в любой момент.
7. Контакты
По вопросам обработки персональных данных вы можете обратиться по адресу: https://цимпи.рф.
